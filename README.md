Project Title : PlaceMate- Placement Analysis and Prediction System

Project Description:

PlaceMate is a comprehensive, full-stack web solution designed to bridge the gap between academic preparation and industry recruitment requirements. By leveraging historical placement trends and machine learning, the platform acts as a continuous career readiness navigator, guiding students from their freshman year through to final-year placements.
